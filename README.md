# csc430-asgn6

## Assignment 6: Elixir


### Some useful tips:
If you have Homebrew, you can easily install Erlang and Elixir with these commands:

**brew install erlang**

**brew install elixir**

### To access the interactions window:

**iex**

(double-click Control C to exit)

### To run this program:

**elixir asgn6.exs**
